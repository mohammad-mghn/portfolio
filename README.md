<h2 id="get-started">
ü§ñ Get Started
</h2>

download zip file, or clone it using git.
```
git clone https://github.com/vito-mohagheghian/portfolio.git
```

then install dependencies using

```
npm install

# or 

yarn dev
```


to setup 

```bash
npm run dev

# or

yarn dev
```
Now click this link [https://localhost:3000](https://localhost:3000)
